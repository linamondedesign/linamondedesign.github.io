Project Overview

쇼핑몰 창업의 문턱을
낮추다

몰파이(Mallpie)는 누구나 코딩 없이 나만의 커머스몰을 만들 수 있는 SaaS 기반 커머스 플랫폼입니다. 비전문가 사용자도 단 몇 분 안에 쇼핑몰을 개설하고 운영할 수 있도록, 모든 설정과 관리 UX를 최대한 단순하고 명확하게 재설계했습니다.

온보딩 플로우 최적화, 대시보드 UX 리뉴얼, 상품 등록 & 관리 시스템 설계 등 사용자가 성공적인 셀러로 성장하는 여정 전체를 디자인했습니다.

#플랫폼 #SaaS #커머스 #UI/UX #대시보드 #온보딩

Design Process

플랫폼을 설계하는 방식

셀러의 성공이 곧 플랫폼의 성공이라는 관점으로, 4단계 접근법을 적용했습니다.

01
셀러 여정 분석
신규 셀러의 온보딩부터 첫 매출까지의 여정을 분석해 핵심 마찰 포인트를 발굴했습니다.
02
정보구조 재설계
복잡한 관리 메뉴를 사용자 멘탈 모델에 맞게 재구조화하고 내비게이션을 단순화했습니다.
03
UI 설계 & 프로토타입
직관적인 대시보드 UI와 인터랙티브 프로토타입을 제작해 팀 내 빠른 피드백을 수렴했습니다.
04
출시 & 성과 검증
베타 테스트 후 데이터 기반으로 온보딩 완료율을 개선하고 서비스를 성공 출시했습니다.
